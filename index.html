webbynode-api by shanesveller

Ruby wrapper library for the WebbyNode XML API using HTTParty

This gem provides a library wrapping the WebbyNode XML API using HTTParty. Full coverage of the API functionality is planned.

Dependencies

httparty (and thus also crack)

Install

gem install --source http://gems.github.com/ shanesveller-webbynode-api
